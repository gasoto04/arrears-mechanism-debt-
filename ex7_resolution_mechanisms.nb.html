6. Paris Club member mapping

The Paris Club (PC) is the group of official bilateral creditors that coordinates sovereign debt restructurings. We identify each member by their IDS counterpart area code (ca_code) and the year they formally joined the PC.

A country’s bilateral debt to a PC member is classified as “officially covered” (subject to the Paris Club mechanism) IF the observation year >= the creditor’s accession year. Debt to all other bilateral creditors (China, India, Gulf states, etc.) is “non-PC” and classified as outside the framework before 2020.

From 2020 onward, the Common Framework (CF) extends to ALL bilateral creditors, so we reclassify non-PC bilateral as covered from that year.

8. Build bilateral components: Paris Club vs. non-PC

We split bilateral debt into two components:

blat_pc: debt owed to Paris Club member countries blat_nonpc: debt owed to all other bilateral creditors (China, India, Gulf, etc.)

Method: - blat_total = WLD aggregate row of DT.DOD.BLAT.CD (total bilateral) - blat_pc = sum of individual creditor-country rows where ca_code is a PC member AND year >= that member’s accession year - blat_nonpc = blat_total - blat_pc (floored at 0 for data consistency)

# Step A: total bilateral per country-year (WLD aggregate row)
blat_total <- ids_long %>%
  filter(series_code == "DT.DOD.BLAT.CD", ca_code == "WLD") %>%
  select(all_of(KEY), blat_total = value)

# Step B: PC bilateral — sum of individual PC-member creditor rows,
#          counting only years in which that creditor was already a PC member
blat_pc <- ids_long %>%
  filter(
    series_code == "DT.DOD.BLAT.CD",
    ca_code %in% pc_mapping$ca_code        # only rows for PC member countries
  ) %>%
  left_join(pc_mapping %>% select(ca_code, accession_year), by = "ca_code") %>%
  filter(year >= accession_year) %>%       # respect each member's join date
  group_by(across(all_of(KEY))) %>%
  summarise(blat_pc = sum(value, na.rm = TRUE), .groups = "drop")

# Step C: non-PC = total minus PC (pmax prevents negative values from data gaps)
bilateral <- blat_total %>%
  left_join(blat_pc, by = KEY) %>%
  mutate(
    blat_pc    = replace_na(blat_pc, 0),
    blat_nonpc = pmax(blat_total - blat_pc, 0)
  )

cat("Bilateral panel rows:", nrow(bilateral), "\n\n")

11. Apply resolution mechanism classification

This is the core step. We assign each row a mechanism label using case_when, then add a boolean covered flag.

Rules (from economist):

Bilateral PC → “Official covered (Paris Club / CF)” — always Bilateral Non-PC → “Official covered (Paris Club / CF)” — from 2020 (Common Framework) → “Official outside framework” — before 2020 Commercial Banks → “Private banks covered (London Club)” — from 1976 → “Private outside framework” — before 1976 Bonds → “Bond debt covered (CACs)” — from 2003 → “Private outside framework” — before 2003 Other Private → “Private outside framework” — always (no mechanism) Multilateral + IMF → “Multilateral / IMF outside” — always (senior creditors)

panel_classified <- panel_long %>%
  mutate(
    mechanism = case_when(
      # --- Official bilateral ---
      creditor_type == "Bilateral PC"                       ~ "Official covered (Paris Club / CF)",
      creditor_type == "Bilateral Non-PC" & year >= 2020   ~ "Official covered (Paris Club / CF)",
      creditor_type == "Bilateral Non-PC" & year <  2020   ~ "Official outside framework",

      # --- Private banks: London Club formed in 1976 ---
      creditor_type == "Commercial Banks" & year >= 1976   ~ "Private banks covered (London Club)",
      creditor_type == "Commercial Banks" & year <  1976   ~ "Private outside framework",

      # --- Bonds: CACs became standard in 2003 ---
      creditor_type == "Bonds"            & year >= 2003   ~ "Bond debt covered (CACs)",
      creditor_type == "Bonds"            & year <  2003   ~ "Private outside framework",

      # --- Other private: no mechanism exists ---
      creditor_type == "Other Private"                     ~ "Private outside framework",

      # --- Multilateral + IMF: senior creditors, outside restructuring ---
      creditor_type %in% c("Multilateral", "IMF Credit")  ~ "Multilateral / IMF outside"
    ),

    # TRUE if this debt is under some restructuring mechanism; FALSE if outside
    covered = mechanism %in% c(
      "Official covered (Paris Club / CF)",
      "Private banks covered (London Club)",
      "Bond debt covered (CACs)"
    )
  )

13. Annual median across countries + renormalize to 100%

Two steps:

Step 1 — Median: For each year × mechanism, take the MEDIAN share across countries. This gives equal weight to each country regardless of debt size. A “typical” country’s debt composition for that year.

Step 2 — Renormalize: Divide each mechanism’s median by the sum of all medians for that year, then multiply by 100. This corrects for the fact that IDS components don’t perfectly sum to the denominator at country level, and ensures the stacked chart always sums to exactly 100%.

We produce three aggregates: LIC-only, MIC-only, All (both groups).

# In geom_area, the LAST factor level sits at the BOTTOM of the stack.
# List Multilateral first (top) and Official covered last (bottom).
mechs_ordered <- c(
  "Multilateral / IMF outside",            # first  → TOP of chart
  "Private outside framework",
  "Official outside framework",
  "Bond debt covered (CACs)",
  "Private banks covered (London Club)",
  "Official covered (Paris Club / CF)"     # last   → BOTTOM of chart
)

# Reusable function: aggregate one group, produce normalized annual data
aggregate_group <- function(data, group_label) {

  # Step 1: median per year × mechanism across countries
  annual <- data %>%
    group_by(year, mechanism) %>%
    summarise(
      median_share = median(share, na.rm = TRUE),
      n_countries  = n_distinct(country_code),
      .groups      = "drop"
    )

  # Ensure every year × mechanism combination is present (fill missing with 0)
  annual <- annual %>%
    complete(
      year      = 1970:2024,
      mechanism = mechs_ordered,
      fill      = list(median_share = 0, n_countries = 0)
    )

  # Step 2: renormalize to 100% per year
  annual <- annual %>%
    group_by(year) %>%
    mutate(

    ) %>%
    ungroup() %>%
    mutate(
      income_group = group_label,
      mechanism    = factor(mechanism, levels = mechs_ordered)
    )

  annual
}

make_chart <- function(annual_data) {
  ggplot(annual_data,
         aes(x = year, y = pct_norm, fill = mechanism)) +
    geom_area(position = "stack", alpha = 0.88) +
    # Vertical dashed lines for structural breaks
    geom_vline(
      data        = transitions,
      aes(xintercept = xintercept),
      linetype    = "dashed",
      color       = "grey20",
      linewidth   = 0.7,
      inherit.aes = FALSE
    ) +
    # Labels for the break lines (positioned at top of chart)
    geom_text(
      data        = transitions,
      aes(x = xintercept + 0.4, y = 97, label = label),
      inherit.aes = FALSE,
      hjust       = 0, vjust = 1,
      size        = 5.0, color = "grey20", lineheight = 0.85
    ) +
    scale_fill_manual(
      values = color_map,
      name   = NULL,
      guide  = guide_legend(nrow = 2, reverse = TRUE)  # reverse so legend matches chart bottom→top
    ) +
    scale_x_continuous(breaks = seq(1970, 2025, 5), expand = c(0, 0)) +
    scale_y_continuous(
      breaks = seq(0, 100, 20),
      labels = function(x) paste0(x, "%"),
      expand = c(0, 0)
    ) +
    labs(x = "Year", y = "Percent of total external debt") +
    theme_minimal(base_size = 20) +
    theme(
      legend.position    = "bottom",
      panel.grid.minor   = element_blank(),
      panel.grid.major.x = element_blank(),
      legend.text        = element_text(size = 12)
    )
}
